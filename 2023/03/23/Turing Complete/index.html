指令集

指令 语法 含义
add add <1> <2> <3> 将寄存器1的值与寄存器2的值相加，结果存到寄存器3中
sub sub <1> <2> <3> 将寄存器1的值与寄存器2的值相减，结果存到寄存器3中
inc inc <1> nop <2> 将寄存器1的值自加1，结果存到寄存器2中
dec dec <1> nop <2> 将寄存器1的值自减1，结果存到寄存器2中
mul mul <1> <2> <3> 将寄存器1的值与寄存器2的值相乘，结果存到寄存器3中
nec inc <1> nop <2> 将寄存器1的值取反，结果存到寄存器2中
div div <1> <2> <3> 将寄存器1的值与寄存器2的值相除（整数），结果存到寄存器3中
and and <1> <2> <3> 将寄存器1的值与寄存器2的值按位与，结果存到寄存器3中
or or <1> <2> <3> 将寄存器1的值与寄存器2的值按位或，结果存到寄存器3中
not not <1> nop <2> 将寄存器1的值按位取反，结果存到寄存器2中
xor xor <1> <2> <3> 将寄存器1的值与寄存器2的值按位异或，结果存到寄存器3中
shl shl <1> <2> <3> 将寄存器1的值左移寄存器2的值，结果存到寄存器3中
shr shr <1> <2> <3> 将寄存器1的值又移寄存器2的值，结果存到寄存器3中
save save <1> to <2> 将寄存器1的值保存到硬盘2的位置
load load <1> to <2> 将硬盘1的位置的值加载到寄存器2上
write write <1> to <2> 将寄存器1的值保存到内存2的位置
read read <1> to <2> 将内存1的位置的值加载到寄存器2上
print print <1> to <2> 将硬盘1值所对应的字符打印到显示屏2位置
imm imm <1> to <2> 将参数1作为值赋给寄存器2
mov mov <1> to <2> 将寄存器1的值移动到寄存器2
pop pop nop nop <1> 弹出栈值至寄存器1
push push <1> nop nop 将寄存器1的值压栈
je je <1> <2> <3> 判断寄存器1的值与寄存器2的值是否相等，若成立，则跳转至寄存器3所示的地址
jne jne <1> <2> <3> 判断寄存器1的值与寄存器2的值是否不相等，若成立，则跳转至寄存器3所示的地址
jl jl <1> <2> <3> 判断寄存器1的值是否小于寄存器2的值，若成立，则跳转至寄存器3所示的地址
jle jle <1> <2> <3> 判断寄存器1的值是否小于等于寄存器2的值，若成立，则跳转至寄存器3所示的地址
jg jg <1> <2> <3> 判断寄存器1的值是否大于寄存器2的值，若成立，则跳转至寄存器3所示的地址
jge jge <1> <2> <3> 判断寄存器1的值是否大于等于寄存器2的值，若成立，则跳转至寄存器3所示的地址
alws alws nop nop <1> 总是跳转至寄存器1所示的地址

其他关键字

关键字 含义
nop/to 占位符，无意义
re0~re15 寄存器0~15
ram 内存，ram+<数字>表示内存地址
hdd 硬盘，hdd+<数字>表示硬盘地址
